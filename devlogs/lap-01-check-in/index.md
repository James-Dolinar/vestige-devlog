# Lap 1 Check-in:

I haven't done any work yet on my project for [Slow Jam 03](https://forum.gamemaker.io/index.php?threads/chequered-flag-slow-jam-03-discussion.122727/) yet, but I've thought about what I want to do and have made some goals for the project.

I've decided that I want to make a game inspired by the GameMaker classic [Seiklus](https://en.wikipedia.org/wiki/Seiklus). The focus will be a simple game in a world that is fun to explore.

## Design Goals:
- Make an interesting open world that is fun to explore
- Incorporate relaxing ambience
- Include the Jam theme while not outright pointing it out

## Learning Goals:
- Learn advanced texture management if I have time
- Learn a bit of shader stuff if I have time

## Other Goals:
 - By the end of the jam, have my game running using the GMRT


I think the most challenging part for me will be to make an interesting world that people actually want to explore. Hopefully I can do it!
